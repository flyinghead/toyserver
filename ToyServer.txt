
TOYSER - Toy Racer Server - LINUX version 9 - 11.17.2000


1. unzip ToyServer.zip -> 2 files : 
	- ToyServer : the server
	- ToyServer.cfg : configuration file

2. copy those files to a directory
3. type this besh command : 
chmod -x ToyServer
4. to launch the server type :
./ToyServer
5. the console should display :
Starting TOY RACER server...
TOY RACER server (version 9) started!

6. now you can type this commands on the server console :
	- quit
	- help
	- stats
	- kill
	- list
	- info
	- high
	- msg
	- refresh
if you type a bad command or just press ENTER key, the help message will be displayed.

quit : shutdown the server
help : display help message
stats: display number of connected clients/games/chat rooms
kill : you can diconnect a player from that server (you will be prompted for the player's ID)
list : you can display the list of players/games/chat rooms/info rooms
info : display more infos on a specified player/game/chat room/info room
msg  : to send an emergency message to ALL the connected players
refresh : force an info room to re-load its text file
high : display highscores

//-------------------------------------------------------------------------
ToyServer.cfg : configuration file

default content is :
TCP_Port:2048
UDP_Port:2049
NbPlayers:512
NbGames:512
NbChats:512
UDP_time_out:7000
client_UDP_time_out:9000
Command_paquets_per_sec:8
Dynamix_paquets_per_sec:6
info_Chat_list_nb:0
info_Chat_list_start:
info_Chat_list_end:


TCP_Port, UDP Port : ports # referenced in Dream Arena authentication server
NbPlayers : max number of players allowed on this server
NbGames/NbChats : max number of games/chat rooms
UDP_time_out/client_UDP_time_out : timeout in UDP game mode (during race) (server/client side)
Command_paquets_per_sec/Dynamix_paquets_per_sec : game parameters
info_Chat_list_nb:0 /info_Chat_list_start: /info_Chat_list_end: : info chat rooms list
to add a chat room : increase field : info_Chat_list_nb, then add a new line between
'info_Chat_list_start:' and 'info_Chat_list_end:' labels
syntax : ChatName,lg,text.txt
where :
	- ChatName is the name of the chat room
	- lg is the language code : 
		- fr : french
		- en : english
		- ge : german
		- sp : spanish
	- text.txt is the path to an ASCII text file containing the messages of the info chat room
		max 100 lines of 256 chars

text.txt sample :
Hello World!
Welcome to the brand new Toy Racer Server!
Have a lot of fun!

LOG files generated by server :

ncToyServerSocket.log
ncToyServerClient.log
ncToyServerChat.log
ncToyServerGame.log

in the server directory.
This files are for debugging.... they may grow !!!

//-------------------------------------------------------------------------
Statistics server file:
ToyServer.sta

Each day (at 00:00) a line will be added to this file :

syntax :
11/5/2000;800;7129;6856;26225;90731;90945;112;14:10:05;95;15:05:00;51;10:08:13
=
date;number of connections;number of games;number of chat rooms;seconds connected;total games duration in sec;total chat rooms duration in sec;Maximum connected players;time of max connected players;max games;time of max game;max chats;time of max chat

//-------------------------------------------------------------------------
Other files :
ToyBestLaps.txt	: best online laps for each vehicles and each race (text format)
ToyServer.lap   : same but binary format
ToyServer.hs    : offline High scores (binary format)
ToyHighScores.txt : same but text format
ToyHighScores.cht : same but text format for High scores info room
